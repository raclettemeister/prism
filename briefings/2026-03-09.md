## 🛠️ TOOLS TO TRY

- **[Agent Safehouse](https://agent-safehouse.dev/)** — macOS kernel-level sandboxing for Claude Code, Codex, and other agents. Wraps `--dangerously-skip-permissions` with a deny-first filesystem model. Your SSH keys, other repos, and `.aws` credentials become invisible to the agent by kernel enforcement.
  - Try it: `curl -fsSL https://raw.githubusercontent.com/eugene1g/agent-safehouse/main/dist/safehouse.sh -o ~/.local/bin/safehouse && chmod +x ~/.local/bin/safehouse` — then alias `claude` to run inside it. Total setup: 10 minutes. Remaining 20 minutes: verify it blocks `cat ~/.ssh/id_ed25519`.
  - Relevance: Every Cursor agent session you run on David Toolkit. Directly reduces the 1%-chance-of-disaster risk on your primary development machine.
  - Signal: HN-featured, tested against Claude Code, Codex, Amp, Gemini CLI, Cursor Agent, Cline, and others. Single-source but technically specific and immediately usable.

- **[pmc — pack-my-code](https://github.com/Water-Run/pack-my-code)** — Single binary that packages your entire codebase into an LLM-ready context file (Markdown or YAML). Follows `.gitignore`, supports include/exclude patterns, outputs tree structure.
  - Try it: Run `pmc . -t -s -o context.md` in your `david-toolkit` repo. Review what Claude sees when you paste that file into a fresh session. 15 minutes.
  - Relevance: Your Cursor sessions on David Toolkit. Useful for onboarding a fresh Claude Code session to the full repo state without manually pointing at files.
